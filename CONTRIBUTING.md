# Contributing to Claude Code Leak Wiki

Thank you for your interest in contributing to this documentation repository!

## How to Contribute

### 1. Report Errors or Missing Information

If you find any inaccuracies or missing details:
- Open an issue with the label `correction`
- Provide the correct information if possible
- Include sources/references

### 2. Add New Discoveries

If you discover new information from analyzing the leaked code:
- Create a pull request
- Include detailed documentation
- Add references and sources

### 3. Improve Documentation

- Fix typos and grammar
- Improve clarity and readability
- Add diagrams or visual aids
- Translate to other languages

### 4. Share Resources

- Add relevant news articles
- Link to helpful tools
- Share analysis blog posts

## Pull Request Guidelines

1. **Clear Title**: Describe what your PR adds/fixes
2. **Detailed Description**: Explain the changes
3. **Source References**: Include links to sources
4. **No Spam**: Ensure content is relevant and valuable

## Style Guide

- Use clear, professional language
- Include code blocks with syntax highlighting
- Add tables for structured data
- Use emojis sparingly and meaningfully

## Questions?

Feel free to open an issue for any questions about contributing.
